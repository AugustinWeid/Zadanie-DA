# Этапы АБ-теста

# 1. Определение целей
**Бизнес-цель:** сократить время доставки, дабы увеличить количество заказов за счёт увеличения качества сервиса.
**Цель эксперимента:** оценить влияние замены транспортного средства на время доставки в одном небольшом городе.

# 2. Определение метрик
**Основная метрика:** среднее время доставки (в минутах).
**Дополнительные метрики:** доля своевременных доставок, отзывы и оценки клиентов, количество заказов на доставщика, расходы на транспорт.
**Метрики качества данных:** сбалансированное распределение групп по дням недели, погодным условиям и объёму заказов.

# 3. Разработка гипотезы
**Нулевая гипотеза:** среднее время доставки в экспериментальной группе (новый транспорт) равно среднему времени в контрольной группе (старый транспорт).
**Альтернативная гипотеза:** среднее время доставки в тестовой группе отличается от контрольной.

# 4. Постановка эксперимента
**Случайное разделение доставщиков** на две группы в одном городе. Предполагается, что эксперимент проведён корректно.
**Единица эксперимента:** доставка заказа.
**Длительность:** достаточная для того, чтобы охватить разные дни недели и погодные условия.
**Контроль:** необходимо убедиться, что группы сопоставимы по всем прочим факторам (погода, день недели, время суток).

# 5. Проведение эксперимента
- Запуск эксперимента в выбранном городе.
- Сбор данных по времени доставки и дополнительным метрикам.

# 6. Анализ результата
**Анализ:**

**Внутренняя валидность**  
   Допустим, что эксперимент проведён корректно: случайное распределение, отсутствие смешивающих факторов, статистическая значимость результата подтверждена. Это значит, что в данном городе изменение действительно привело к снижению времени доставки.

**Внешняя валидность**  
   Тест проводился только в одном небольшом городе, а это значит, что этот город может сильно отличаться от других населённых пунктов:
   - По трафику,
   - По инфраструктуре,
   - По климату,
   - По поведению людей.
   Это значит, что результаты нельзя просто взять и перенести на все города страны. Эксперимент применим лишь для небольших городов, похожий на тот, который тестировали.

**Интерпретация и дальнейшие шаги**  
   - Потенциал идеи есть, так как есть положительный эффект, но этого недостаточно для применения на всю страну.
